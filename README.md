# Director Motorcycle Cinema

Demo: https://holynova.github.io/director-motorcycle-cinema/

GitHub: https://github.com/holynova/director-motorcycle-cinema

A static cinema-style gallery of 50 director-inspired motorcycle city images.

## Features

- 50 image cards with Chinese / English director names
- Dark private-screening-room visual style
- Category filters: 全部 / 华语导演 / 亚洲大师 / 世界电影
- Equal-size 3:2 card image areas
- Full-image display with `object-fit: contain` and no cropping
- Lightbox preview
  - Desktop: previous / next buttons and keyboard ← / → navigation
  - Mobile: swipe left / right navigation
- Zero framework dependencies; deploys as static GitHub Pages files

## Deployment

This site is published with GitHub Pages from the repository root.
